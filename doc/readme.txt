有四种模式,用背面的按键切换
[ 4.01 ] 一般模式,连上wifi，把温度上传到服务器,并显示服务器下发的内容， 温度，美元汇率.....
[AP120]  AP模式，此时，会有一个disp的无线接入点,
         手机或者电脑可以联上disp，浏览网址aaa.com进行wifi设置
[ OTA ]  OTA模式，如果wifi已经设好，屏幕会显示ip，可以用arduino开发环境进行升级,或者浏览器浏览ip，进行设置
[ OFF ]  关机模式,off模式下不能充电

以上是一般使用的说明

    内置300mah的锂电池

    OTA模式下，需要连上wifi，会显示自身IP，用arduino会自动发现设备，或者用espota.py手动升级
背面的6个引脚，是TTL串口和I2C扩展接口，从上到下依次是3.6V,3.3V,SDA,SCL,GND,RX,TX
如果是5个引脚， 就是缺少3.6V,

3.6V是直接接内置锂电电池的，可以直接并接外置锂电电池进行增容。 内置锂电池300mah,可以用40天。

故障解决:
    如果按键后，灯能闪，但屏幕没有变化，可能有2种情况，第一种是LCD死机，可以短路一下背面的3.3V和GND复位一下，然后插上充电线;如果无效，那就是程序乱了，需要用GND,RX,TX进行线刷。
线刷，要在arduino开发环境进行
